# my-project

## Prerequisites

*   **Java SDK**: Version 25 or higher


## Build Instructions

This project uses Maven for dependency management and building.

To build the project and run tests, use the following command:

On Windows:

```shell
.\mvnw.cmd verify
```

On Mac/Linux:

```shell
./mvnw verify
```
